# AIWD_LAB1
Artificial Intelligence in Web Development Laboratory 1

Description

This experiment demonstrates the creation of an Online Event Registration & Promotional Portal using HTML5. The webpage displays event information using HTML5 media elements such as image, video, and audio. It also includes an event registration form where users can enter their details using various HTML5 input fields such as text, email, date, radio buttons, select menus, textarea, and file upload. This experiment shows how multimedia and interactive form elements can be integrated into a single webpage.
